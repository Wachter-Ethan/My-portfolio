Projet : Site cinema

Durant ce projet, je suivait l'idée de mon père qui était de créer un site sur lequel des classements de films étaient souvant mis à jour. Ces classements peuvent être par catégorie ou par meilleur film du momment ...
De plus, l'objectif de ce site est de préciser si le film est disponible ou non sur différentes applications de streaming tel que Netflix ou Disney +.
Ce site est mon premier gros projet mais celui-ci est toujours en cours de développement.

Ce site est composé de plusieurs pages. Celle-ci est la page principale, elle est sépareée en 3 parties :

- Le film de l'année
- Les meilleurs films des 10 dernières années
- Les différentes catégories de films.

Voici le premier classement sur les 10 dernières années, il se présente sous la forme d'un tableau avec le film et sa note.

Toutefois, vous pourrez observer que ce site ne comporte pas de texte mais uniquement des notations comme des commentaires. Cela est dû au fait que je souhaitais d'abord valider le design, ma prochaine étape sera de remplir le texte.

Afin de proposer un maximum de choix, j'ai mis en place une partie dans laquelle on peut choisir la catégorire que l'on souhaite afin d'obtenir le classement la concernant.

J'ai décidé de présenter le site de cette façon afin de facilité la navigation grâce au classement par catégorie.

Projet : Jeu Pyxel

Ce projet était un projet de groupe réalisé dans le cadre d'un travail noté dans ma spécialité NSI (numérique et science informatique) en classe de première.
Ce projet réalisé en groupe de deux avait pour objectif de faire un vaisseau représenté par un carré bleu, qui tirait sur une cible représentée par un carré blanc. De plus, nous devions faire augmenter le score de 1 à chaque fois que la cible était touchée. Le langage de programation imposé était le python avec la library Pyxel.

Maintenant, Nous pouvons observer le code du jeu. Celui-ci s'etend sur plusieurs dizaines de lignes. Chaqu'un des élément est modélisé par une fonction qui agit constament durant le jeu.
Toutefois, des actions telque se déplacer, tirer ... sont codés avec des conditions (if ... else ...).

Finalement, ce projet de groupe fut très utile pour nous permettre de mieux maitriser le langage python mais également et nous à permis d'évoluer en travail d'équipe, une compétence essentielle dans tout métier !

Projet : Serveur Flask local

Ce projet était un projet en groupe noté durant l'année de première dans laquelle nous devions mettre en place un serveur Flask en local avec python pour héberger un site suivant un cahier des charges. Ce site devait contenir 5 questionnaires différents et chaqu'un devaient contenir 5 questions à choix multiples. La contrainte était que nous devions permettre à l'utilisateur de choisir le QCM de son choix et qu'il ne quitte jamais la page.

Le code ci-contre est le code python qui permet de générer le serveur Flask. Ce code n'est pas très complexe et fait appel à des fonctions qui créent des chemins pour accéder à différentes pages. De plus, nous avons utilisé "render_template", ceci permet de récupérer des documents telque des fichiers html dans un dossier nommé "templates".

Finalement, sur cette image, il s'agit de la page principale du site. Ici nous pouvons choisir 5 catégories. Nous avons également ajouté une annimation svg dans le fond comme nous le demandait le cahier des charges. Toutefois, contrairement au reste du site, nous nous sommes aidés d'un autre site pour récupérer l'annimation svg. Il en est de même pour l'annimation de fond de ce site.
Nous pouvons également accéder aux autres pages à partir de celle-ci grâce au bouton présent au bas de cette page.

Projet : jeu SNAKE en html/css/javascript

Ce projet à vu le jour grâce à un concours au sein de mon lycée. Ce concours consistait à créer un code avec le langage de notre choix. Je me suis donc basé sur le jeu SNAKE que j’avais codé précédemment en python avec la library pyxel, que j’ai recodé avec le langage html, css et javascript.
Quand on lance la page avec le jeu on tombe sur une interface comportant le plateau de jeu, les scores et niveaux, une zone qui va accueillir des messages visant à personnaliser le jeu, et enfin les touches à utiliser pour faire se déplacer le serpent.

Dans ce projet, l'essentiel du code est en javascript. Cela est dû au grand nombre de fonctions nécessaires et de calcul à faire ( scores, niveau, placement du serpent, placement des cibles … ). Ce code fait appel à des modifications constantes entre les positions précédentes et les suivantes. De plus, ce code se base sur un style très similaire à la library pyxel du langage python, car on utilise des fonctions qui s'effectuent en continu.

Je pense faire encore évoluer ce jeu en ajoutant quelques fonctionnalités comme par exemple le game over, styliser le serpent…
