Projet : Jeu Pyxel

Ce projet était un projet de groupe réaliser dans le cadre d'un travail noté durant ma spécialité NSI (numérique et science informatique) en classe de première.
Ce projet réaliser en groupe de deux avait pour objectif de faire un vaisseau représenté par un carré bleu, tiré sur une cible représenté par un carré blanc. De plus, nous devions faire augmenté le score de 1 à chaque fois que la cible était touché. Le langage de programation imposé était le python avec la library Pyxel.

Maintenant, Nous pouvons observé le code du jeu. Celui-ci s'etend sur plusieur dizaine de ligne. Chaqu'un des élément est modéliser par une fonction qui agit constament durant le jeu.
Toutefois, des actions tel que se déplacer, tiré ... sont codé avec des conditions (if ... else ...).

Finalement, ce projet de groupe fut très utile pour nous permètre de mieux métriser le langage python mais également pour évoluer dans le travail d'équipe, une compétence essentiel dans tout métier !
