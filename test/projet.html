Projet : Serveur Flask local

Ce projet était un projet en groupe noté durant l'année de première dans lequel nous devions metre en place un serveur Flask en local avec python pour éberger un site suivant un cahier des charges. Ce site devait contenir 5 questionnaires différent et chaqu'un devaitcontenir 5 questions à choix multiple. La contrainte était que nous devions permetre à l'utilisateur de choisir le QCM de sont choix et que l'on ne quite jamais la page.

Le code ci-contre est le code python qui permet de généré le serveur Flask. Ce code n'est pas très complexe et fait appelle à des fonction qui créent des chemins pour accéder à différente page. De plus, nous avons utiliser "render_template", ceci permet de récupérer des documents tel que des fichiers html dans un dossier nommé "templates".

Finalement, sur cette image, il s'agit de la page principale du site. Ici nous pouvons choisir 5 catégories. Nous avons également ajouter une annimation svg dans le fond comme nous le demander le cahier des charges. Toutefois, contrairement au reste du site, nous nous sommes aider d'un autre site pour prendre l'annimation svg. Il en est de même pour l'annimation de fond de ce site.
Nous pouvons également accéder au autre page à partir de celle-ci grace au bouton présent au bas de cette page.
